TO USE IN ECLIPSE:
1. Open Eclipse
2. In the Menu Bar select File > New > Java Project
3. Uncheck 'Use Default Location'
4. Click "Browse..."
5. Locate and select the folder either extracted from this .zip or downloaded off of GitHub
6. Click Finish
7. Right Click Tab2PDF in the Package Explorer
8. Select Properties
9. Select 'Java Build Path'
10. Click 'Add Library...'
11. Select JUnit and click Next
12. Select 'JUnit 4' from the dropdown and click Finish
